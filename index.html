Custom IP For Minecraft

Follow These Steps

1. Take your server IP and put it in the search bar.
2. Click on "Copy IP Address".

Create Your Custom IP

1. Write your Email ID in the EMAIL bar.
2. Paste the copied IP Address in the SERVER IP bar.
3. Enter your server port in the PORT bar.
4. Write your desired custom IP.
5. Select a domain in the ZONE/DOMAIN bar.
6. Click the CREATE button.
